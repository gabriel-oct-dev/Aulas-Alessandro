# Engenharia Cognitiva

## Interface Homem-Máquina

A engenharia cognitiva é uma área que estuda como os humanos interagem com máquinas, considerando aspectos fundamentais que influenciam essa interação. Esses aspectos incluem:

- **Atenção**: Capacidade de focar em elementos importantes da interface.
- **Percepção e Reconhecimento**: Identificação de padrões e símbolos apresentados.
- **Memória**: Retenção de informações para uso durante a interação.
- **Aprendizado**: Adaptação ao funcionamento do sistema.
- **Leitura, Fala e Audição**: Comunicação entre o usuário e o sistema.
- **Tomada de Decisão**: Escolha de ações com base nas informações disponíveis.
- **Planejamento**: Organização de etapas para alcançar um objetivo.
- **Raciocínio**: Processamento lógico para resolver problemas.

---

## Modelos Cognitivos

Os modelos cognitivos são ferramentas que ajudam a compreender como os usuários interagem com sistemas. Eles incluem:

- **Modelo do Design**: Representa a visão do designer sobre como o sistema deve funcionar, incluindo suas funcionalidades e aparência.
- **Modelo de Tarefas**: Descreve as etapas que o usuário precisa realizar para atingir um objetivo específico.
- **Modelo de Uso**: Reflete as expectativas do usuário sobre o funcionamento do sistema, baseadas em sua experiência prévia.
- **Imagem do Sistema**: É a interface real apresentada ao usuário, que deve ser intuitiva e alinhada às expectativas do usuário.

## Golfo de Execução e Avaliação

O **Golfo de Execução e Avaliação** descreve o ciclo de interação entre o usuário e o sistema, dividido em duas partes:

### Golfo de Execução

Representa as etapas que o usuário realiza para executar uma ação no sistema:

```plaintext
Usuário
   o
  /|\
   |
  / \
 | Formulação da Intenção | Sequência de Ações | Execução |
 |          1             |         2          |    3     |
       |
       v
     Sistema
```

### Golfo de Avaliação

Representa as etapas que o sistema realiza para fornecer feedback ao usuário:

```plaintext
     Sistema
       o
      /|\
       |
      / \
 | Percepção | Interpretação | Avaliação |
 |     1     |       2       |     3     |
       |
       v
     Usuário
```
